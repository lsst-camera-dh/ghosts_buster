# lsst_ghost_buster
Tool based on Batoid (ray tracing simulation) for the mitigation of ghosts in LSSTComCam.

*Note : for notebooks/comcam_ghosts, you need to run at USDF (notebooks/comcam_ghosts/Fit\* can also be run at CC-IN2P3)*
